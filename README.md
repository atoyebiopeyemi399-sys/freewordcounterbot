# Free Word Counter Bot 🤖

A powerful Telegram bot for counting words, characters, lines, and more. Completely free and open source.

## Features ✨

- 📊 **Word Counting** - Count words in any text
- 🔤 **Character Counting** - Count characters with and without spaces
- 📏 **Line Counting** - Count lines of text
- 🌐 **Language Detection** - Detects English, Arabic, Chinese
- 📈 **User Statistics** - Track your usage
- 🎯 **Fast & Private** - No text is stored permanently

## Commands 📝

- `/start` - Welcome message with buttons
- `/help` - Show help and usage
- `/wordcount <text>` - Count words in specific text
- `/charcount <text>` - Count characters in specific text
- `/stats` - View your usage statistics
- `/about` - About this bot

## Deployment 🚀

### Deploy on Railway

1. Fork this repository
2. Create a new project on Railway
3. Connect your GitHub repository
4. Add environment variable: `TELEGRAM_BOT_TOKEN`
5. Deploy!

### Deploy on Vercel

1. Fork this repository
2. Create a new project on Vercel
3. Connect your GitHub repository
4. Add environment variable: `TELEGRAM_BOT_TOKEN`
5. Deploy!

### Local Development

1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Create `.env` file with `TELEGRAM_BOT_TOKEN=your_token_here`
6. Run: `python bot.py`

## Environment Variables 🔑

- `TELEGRAM_BOT_TOKEN` - Your bot token from @BotFather
